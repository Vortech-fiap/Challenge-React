1. Funes - Crescimento de Seguidores de uma Jogadora

Nesta seo, o objetivo utilizar uma funo exponencial para modelar o crescimento de uma jogadora nas redes sociais. Esse tipo de funo ideal para representar fenmenos que crescem a uma taxa constante, como a popularidade de um atleta em ascenso.

A Funo Exponencial

A funo proposta : $$f(t)=5000\cdot(1,08)^t$$

  • $f(t)$ representa o nmero de seguidores da jogadora em um determinado momento.
  • $t$ representa o tempo, medido em semanas.
  • $5000$ o nmero inicial de seguidores da jogadora (quando $t=0$).
  • $1,08$ a taxa de crescimento semanal. O valor de 1,08 indica que o nmero de seguidores aumenta em 8% a cada semana $(1 + 0,08)$.

Exemplo Prtico: Projeo de Seguidores

Para ilustrar o uso da funo, podemos calcular o nmero de seguidores em diferentes momentos:

  • Semana 0 (incio): $$f(0)=5000\cdot(1,08)^0 = 5000\cdot1=5000$$
  • Semana 5: $$f(5)=5000\cdot(1,08)^5 \approx 5000\cdot1,47\approx7350$$
  • Semana 10: $$f(10)=5000\cdot(1,08)^{10} \approx 5000\cdot2,16\approx10800$$

Grfico do Crescimento de Seguidores

Para visualizar o comportamento da funo de crescimento de seguidores, utilizamos o seguinte cdigo Python. Ele gera um grfico de linha, que a melhor forma de representar a progresso exponencial ao longo das semanas.

O grfico mostra como o nmero de seguidores, que comea em 5.000, aumenta de forma acelerada, reforando a natureza da funo exponencial.

2. Limites - Saturao do Engajamento

Aqui usamos o limite para mostrar que o crescimento da nossa jogadora nas redes sociais no dura para sempre. Embora a funo que usamos para modelar o crescimento seja exponencial e tenda ao infinito, na realidade, a popularidade dela acaba se estabilizando.

Isso acontece porque o alcance tem um "teto". Matematicamente, representamos essa ideia da seguinte forma:

$$\lim_{t \to \infty} f(t) = L$$

Onde $L$ o valor mximo que o nmero de seguidores tende a alcanar. A ideia que a velocidade do crescimento diminui com o tempo at o perfil da jogadora ficar mais estvel.

Grfico Saturao do Engajamento

O grfico a seguir, gerado pelo cdigo, mostra a curva de crescimento se aproximando de um valor mximo. A linha pontilhada vermelha representa esse limite de saturao, ou seja, o "teto" de seguidores que a jogadora tende a alcanar a longo prazo.

import numpy as np
import matplotlib.pyplot as plt

# Define a funo logstica para modelar o crescimento com saturao
# L: Limite superior (valor mximo de seguidores)
# k: Taxa de crescimento
# t0: Ponto central da curva
def logistic_growth(t, L, k, t0):
    return L / (1 + np.exp(-k * (t - t0)))

# Parmetros para a funo
L = 25000  
k = 0.5  
t0 = 10    

# Gera valores para o tempo (t) em semanas, de 0 a 40 para mostrar a saturao
t_values = np.linspace(0, 40, 100)

# Calcula o nmero de seguidores com a funo logstica
followers = logistic_growth(t_values, L, k, t0)

# Cria o grfico
plt.figure(figsize=(10, 6))
plt.plot(t_values, followers, color='navy', label='Crescimento Logstico')

# Adiciona uma linha horizontal para representar o limite
plt.axhline(y=L, color='red', linestyle='--', label=f'Limite de Saturao (L = {L})')

# Adiciona ttulo e rtulos
plt.title('Modelo de Crescimento de Seguidores com Saturao')
plt.xlabel('Tempo (semanas)')
plt.ylabel('Nmero de Seguidores')
plt.grid(True)
plt.legend()

3. Derivadas - Velocidade Instantnea em Campo

Nesta parte, usamos derivadas para analisar o desempenho fsico da jogadora. O conceito de derivada nos permite calcular a taxa de variao de uma funo em um ponto exato, sendo bom para descobrir a velocidade instantnea de uma atleta durante um jogo.

Se temos uma funo que descreve a posio da jogadora em campo em um determinado momento, a derivada dessa funo nos d a velocidade dela naquele instante.

A funo que usamos para a posio da jogadora, medida em metros, $s(t)$, onde $t$ o tempo. Para encontrar a velocidade instantnea, $v(t)$, ns derivamos a funo de posio em relao ao tempo.

A frmula a seguinte: $$v(t) = \frac{ds}{dt}$$

Exemplo Prtico:

Para ilustrar, imagine que a posio de uma jogadora em um sprint pode ser modelada pela funo $s(t) = 0,5t^2 + 5t$, onde $t$ o tempo em segundos e $s$ a distncia percorrida em metros.

Para encontrar a velocidade instantnea da jogadora, calculamos a derivada da funo de posio: $$v(t) = \frac{d}{dt}(0,5t^2 + 5t) = 1t + 5$$

Isso significa que, a cada segundo, a velocidade dela muda. Por exemplo, no instante $t=3$ segundos, a velocidade instantnea seria: $$v(3) = 1(3) + 5 = 8 \text{ m/s}$$

Com isso, conseguimos analisar a capacidade de acelerao da jogadora em momentos do jogo.

Grfico da Velocidade Instantnea

Para ilustrar o conceito de derivada como velocidade instantnea, o cdigo a seguir cria um grfico que relaciona a posio da jogadora com sua velocidade.

Nossa funo para a posio da jogadora : $s(t) = 0,5t^2 + 5t$

Para encontrar a velocidade, que a taxa de mudana da posio, calculamos a derivada dessa funo: $$v(t) = \frac{d}{dt}(0,5t^2 + 5t) = 1t + 5$$

No grfico, a curva azul representa a posio da jogadora, e a linha verde pontilhada mostra a velocidade dela ao longo do tempo. Para exemplificar a velocidade em um instante exato, calculamos a velocidade no ponto $t = 3$ segundos: $$v(3) = 1(3) + 5 = 8 \text{ m/s}$$

A linha pontilhada vermelha a reta tangente, cuja inclinao exatamente 8. Essa reta visualiza a velocidade instantnea da jogadora naquele momento, mostrando como a inclinao da curva de posio corresponde ao valor da velocidade.

import numpy as np
import matplotlib.pyplot as plt

# Define a funo de posio da jogadora
def position_function(t):
    return 0.5 * t**2 + 5 * t

# Define a funo de velocidade (derivada da funo de posio)
def velocity_function(t):
    return t + 5

# Cria um intervalo de tempo para o grfico
t_values = np.linspace(0, 10, 100)

# Calcula os valores de posio e velocidade para o intervalo de tempo
s_values = position_function(t_values)
v_values = velocity_function(t_values)

# Escolhe um ponto especfico para mostrar a velocidade instantnea (e.g., t=3)
t_point = 3
s_point = position_function(t_point)
v_point = velocity_function(t_point)

# Cria o grfico
plt.figure(figsize=(12, 8))
plt.plot(t_values, s_values, color='navy', label='Posio da Jogadora, $s(t)$')
plt.plot(t_values, v_values, color='green', linestyle='--', label='Velocidade, $v(t)$')

# Adiciona a linha de velocidade instantnea no ponto t=3
tangent_line = v_point * (t_values - t_point) + s_point
plt.plot(t_values, tangent_line, color='red', linestyle=':', label='Velocidade Instantnea em t=3')
plt.scatter(t_point, s_point, color='red', marker='o', s=100, zorder=5)

# Adiciona ttulo e rtulos
plt.title('Anlise de Desempenho: Posio e Velocidade da Jogadora')
plt.xlabel('Tempo (segundos)')
plt.ylabel('Distncia (metros) e Velocidade (m/s)')
plt.legend()
plt.grid(True)
plt.tight_layout()

4. Integrais: Clculo da Distncia Total

Enquanto a Derivada (analisada na Sprint 3) foi crucial para encontrar a velocidade mxima a exploso da atleta , ela no mede a resistncia ou o volume de jogo ao longo da partida inteira.

Para uma anlise de desempenho completa, conclumos que fundamental validar a "distncia total percorrida por uma jogadora na partida". Este um KPI (Key Performance Indicator) crtico usado por analistas tticos e preparadores fsicos para avaliar o condicionamento e a contribuio da atleta.

Aqui, a contribuio do Clculo direta: a Integral Definida a ferramenta que nos permite calcular essa distncia. Ao integrar a funo de velocidade, $v(t)$, ao longo do tempo de jogo (de 0 a 90 minutos), calculamos a "rea sob a curva", que representa exatamente a distncia total acumulada.

A integral definida da funo de velocidade $v(t)$ em um intervalo de tempo $[a, b]$ nos d o deslocamento total nesse perodo. Em termos grficos, isso representado pela "rea sob a curva" da funo de velocidade.

  • Funo (Modelo Terico): $v(t) = 1t + 5$
  • Integral Analtica (Terica): A distncia total terica percorrida nos primeiros 10 segundos : $$D = \int_{0}^{10} (t + 5) dt = \left[ \frac{1}{2}t^2 + 5t \right]_{0}^{10}$$ $$D = \left( \frac{1}{2}(10)^2 + 5(10) \right) - (0) = 50 + 50 = 100 \text{ metros}$$

from sympy import Symbol, integrate as sympy_integrate

# 1. Definio da Funo 
# v(t) = 1t + 5
def v(t):
    return 1 * t + 5

# 2. Mtodo da Soma de Riemann 
#    (Calcula a rea usando retngulos pela esquerda)
def soma_riemann(f, a, b, n):
    dx = (b - a) / n
    x_pontos_esquerda = np.linspace(a, b - dx, n)
    area = np.sum(f(x_pontos_esquerda) * dx)
    return area

# 3. Mtodo dos Trapzios 
#    (Calcula a rea usando np.trapezoid para evitar DeprecationWarning)
def metodo_trapezios(f, a, b, n):
    x_pontos = np.linspace(a, b, n + 1) # n+1 pontos para n subintervalos
    y_pontos = f(x_pontos)
    area = np.trapezoid(y_pontos, x_pontos)
    return area

# 4. Clculo da rea Exata
def calcular_area_exata(a, b):
    t_sym = Symbol('t')
    f_sym = 1 * t_sym + 5
    
    # Integral simblica
    area_exata_sym = sympy_integrate(f_sym, (t_sym, a, b))
    
    # Converte para float para clculos de erro
    return float(area_exata_sym)

if __name__ == "__main__":
    
    print("Comparao de Mtodos ")
    
    a, b = 0, 10  # Intervalo do problema: [0, 10] segundos
    area_exata = calcular_area_exata(a, b)
    
    print(f"Funo: v(t) = t + 5")
    print(f"Intervalo: [{a}, {b}]")
    print(f"Distncia Total Exata : {area_exata:.2f} metros\n")
    
    # Lista de 'n' para testar 
    n_values = [5, 10, 50, 100]
    
    resultados_comparativos = []
    
    for n in n_values:
        # Clculo por Riemann
        area_riemann = soma_riemann(v, a, b, n)
        erro_riemann = abs((area_riemann - area_exata) / area_exata) * 100
        
        # Clculo por Trapzio
        area_trapezio = metodo_trapezios(v, a, b, n)
        erro_trapezio = abs((area_trapezio - area_exata) / area_exata) * 100
        
        resultados_comparativos.append((n, area_riemann, erro_riemann, area_trapezio, erro_trapezio))

    df_comp = pd.DataFrame(resultados_comparativos, columns=[
        'n (Intervalos)', 
        'Riemann (Distncia)', 'Erro Riemann (%)',
        'Trapzio (Distncia)', 'Erro Trapzio (%)'
    ])
    
    print(df_comp.to_string(index=False))
    print("\nPerceba como o erro diminui com mais intervalos (n).")
